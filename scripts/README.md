### Homebrew: package manager per macOS
- https://brew.sh  
- Aprire il terminale 
- Eseguire il codice presente sulla homepage di Homebrew
  
In caso di errore provare: 
```
echo "export PATH=/opt/homebrew/bin:$PATH" >> ~/.zshrc
```
